Diet Fry/Steam Chef Mode

  • Press STOP/Cancel.

  • Press Diet Fry/Steam Chef until a desired code (1, z, 3, 4, s) appears in the display.

    • 1: Diet Fry

    • z: Salads

    • 3: Soups

    • 4: Continental

    • s: Steam Chef

  • Select a desired menu code by using More/Less, and press START/Select.

  • Select a desired weight by using More/Less, and press START/Select.

    • During cooking you can turn the dial to adjust the cooking time.
